# VEX-OSINT

OSINT Intelligence Tool by Vexyron

## Features

- IP Lookup (ip-api.com)
- Phone Lookup (veriphone.io)
- Email Lookup (hunter.io)
- Port Scanner
- Whois Lookup
- DNS Resolver
- Metadata Scanner

## Installation

pip install -r requirements.txt
python main.py
